# 📊 Personal Finance Analytics: Spending & Savings Optimization

## 📌 Problem Statement

Managing personal finances effectively requires a clear understanding of income trends, spending patterns, and savings behavior. Without structured analysis, it becomes difficult to identify inefficiencies and optimize financial decisions.

This project aims to analyze financial data and provide actionable insights to improve savings and overall financial health.

---

## 🎯 Objectives

* Track income, expenses, and savings over time
* Identify major spending categories
* Evaluate savings performance against target benchmarks
* Provide actionable recommendations for financial optimization

## 📊 Key Insights

* **40% expenses → Rent** → High fixed cost
* **Groceries + EMIs = ~45%** → Major essential spending
* **Savings rate ~22%** → Below ideal (25–30%)
* **71% savings in Mutual Funds** → Low diversification
* **Savings dip in 2020** → Expense-income imbalance
* **Income growth unstable** → Impacts savings efficiency

---

## 💡 Recommendations

* Reduce discretionary spending (shopping + leisure) by **5–8%**
* Improve savings rate to **25%+** for better financial stability
* Diversify investments beyond mutual funds
* Monitor fixed costs (rent, EMIs) to improve flexibility
