Personal Finance Analytics: Spending & Savings Optimization

Problem Statement:
Managing personal finances effectively requires understanding spending patterns, savings behavior, and income trends. This project aims to analyze financial data to identify inefficiencies and optimize savings.

Objectives:
Track income, expenses, and savings
Identify major spending categories
Evaluate savings performance vs target
Provide actionable financial insights

Key Insights:
40% expenses → Rent → High fixed cost
Groceries + EMIs = ~45% → Major essential spend
Savings rate ~22% → Below ideal (25–30%)
71% savings in Mutual Funds → Low diversification
Savings dip in 2020 → Expense-income imbalance
Income growth unstable → Impacts savings efficiency

Recommendations:
Reduce discretionary spending by 5–8%
Improve savings rate to 25%+
Diversify investments beyond mutual funds

Tools Used:
Power BI
Excel

What I Learned:
Importance of KPI-driven dashboards
